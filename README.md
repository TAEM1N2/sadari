# 사다리 게임 (신나게 아이스크림 사기)

GitHub Pages에 올린 페이지를 휴대폰/PC에서 **앱처럼 설치(PWA)** 해서 사용할 수 있습니다.

## 설치 방법

### 1) Android (크롬)
1. GitHub Pages 주소로 접속
2. 브라우저 메뉴(⋮) → **앱 설치** 또는 **홈 화면에 추가**
3. 홈 화면에 아이콘이 생성됨 → 아이콘으로 실행

### 2) iPhone / iPad (Safari)
1. GitHub Pages 주소로 접속
2. 하단 공유 버튼(□↑) → **홈 화면에 추가**
3. 홈 화면에 아이콘이 생성됨 → 아이콘으로 실행

### 3) PC (Chrome / Edge)
1. GitHub Pages 주소로 접속
2. 주소창 오른쪽 **설치 아이콘** 클릭  
   (또는 메뉴 → **앱 설치**)
3. 바탕화면/시작 메뉴에 앱이 추가됨

## 사용 방법

1. **참가자** 입력  
   - 쉼표(,)로 구분해서 입력  
   - 예: `태민, 찬우, 지용, 윤석, 지우`
2. **꽝 인원** 입력
3. **사다리 생성** 클릭
4. 아래 중 원하는 실행 선택
   - **전체 실행**: 순서대로 결과 애니메이션
   - **빠른 결과보기**: 즉시 결과

### 세팅 창
- 상단 ⚙️ **세팅** 버튼 클릭
- 소리 음량 조절, 참가자/꽝 인원 수정 가능
- **사다리 속도** 바에서 애니메이션 속도 조절

## 참고
- 설치된 앱은 **웹앱(PWA)** 입니다. 앱스토어 설치가 아니라 **홈 화면/바탕화면에 추가되는 형태**입니다.
- 업데이트는 GitHub Pages에 최신 파일을 올리면 자동 반영됩니다.
